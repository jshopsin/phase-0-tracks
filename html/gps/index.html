What is the purpose of HTML5 semantic tags? Which ones do you see as most useful?

HTML5 semantic tags are useful for text readers that help people with disabilities "view" the site. By adding semantic tags, it is adding meaning to the content so the text reader can add emphasis, breaks, etc. to how the text is read. I personally like using the strong tag to add bold or a strong voice. I can also see a lot of use in the article tag to create sections of content with their own headers.

What is the order the browser uses to display elements on a page?

The default display of elements from the html file on a page is in the order it is written, and all stacked. CSS can be used to create in-line block displays so that items can sit next to each other in a window. CSS can also change the position of elements, but this is probably not the best practice if you are able to just change the order in the html file.
